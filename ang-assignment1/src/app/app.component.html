Functionality Requirements


  • An array of objects is passed as a input to the user-details component, where each object contains information about a person. The object has 5 properties:
    name: Name of the person. [STRING]
    designation: His current designation[STRING]
    university: University from which he graduated. [STRING]
    city: Residing city. [STRING]
    imgUrl: Image of person
  • There is an input field for the person name where the user can type the name of a person to search the data of a particular person. (person name is case-insensitive.)
  • If data exists for the typed input, render the person details
    If no data exists for the typed input, do not render the person details but instead render No Results Found
  • At component render, since nothing is typed, do not render above 2 divs.
